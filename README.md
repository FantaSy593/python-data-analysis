Учебные скрипты по анализу данных на Python

В данном репозитории собраны примеры моих учебных скриптов, демонстрирующие базовые навыки работы с данными, автоматизации вычислений и поиска аномалий с использованием библиотек Pandas и NumPy.

Описание скриптов

1. script1_data_cleaning.py — Предобработка и очистка данных
- Загрузка структурированных данных (эмуляция логов).
- Автоматическое удаление дубликатов строк.
- Обработка и заполнение пропущенных значений с использованием средних показателей.

2. script2_metrics_aggregation.py — Расчет продуктовых метрик
- Группировка данных интернет-магазина по категориям товаров.
- Агрегация метрик: подсчет общего количества заказов, суммарной выручки и среднего чека для каждого сегмента.
- Сортировка итогового аналитического отчета.

3. script3_anomaly_detection.py — Статистический поиск аномалий
- Анализ числовых выборок на примере времени сессии пользователей.
- Расчет математического ожидания и стандартного отклонения выборки.
- Реализация алгоритма поиска критических отклонений и выбросов в данных.

4. script4_vectorized_calculations.py — Векторизованные вычисления
- Работа с массивами данных без использования медленных циклов.
- Безопасный расчет конверсии с обработкой деления на ноль через метод numpy.where.
- Фильтрация и выделение сегментов пользователей с высокой активностью.

Используемый стек технологий
- Python 3
- Pandas — работа с датафреймами, агрегация, очистка данных
- NumPy — векторизованные вычисления, математическая статистика
